Pratique agora os 3 pilares do JavaScript.

Nível: Iniciante

1. O Guardador de Nomes

A Explicação: As variáveis (let e const) são como etiquetas em caixas. Se você quer que o computador lembre de algo, você precisa dar um nome para esse dado.

🎯 O Desafio: Crie uma variável para o nome de um carro e outra para o preço dele. Depois, mostre uma frase dizendo: "O carro [nome] custa [preço]".
// Código para testar no console:
let carro = "Ferrari";
let preco = 1000000;
console.log("O carro " + carro + " custa R$" + preco);
Nível: Intermediário

2. A Máquina de Somar

A Explicação: Funções são como "receitas de bolo". Você escreve uma vez e usa sempre que precisar, apenas chamando o nome dela.

🎯 O Desafio: Crie uma função chamada somar que recebe dois números e mostra o resultado em um alerta na tela.
// Código para testar no console:
function somar(n1, n2) {
  let resultado = n1 + n2;
  alert("A soma é: " + resultado);
}
// Para usar, digite: somar(10, 5);
Nível: Mestre

3. O Verificador de Idade

A Explicação: O if (Se) serve para o computador decidir o que fazer. Se uma condição for verdadeira, ele faz uma coisa; se for falsa (else), ele faz outra.

🎯 O Desafio: Crie um código que verifique se uma pessoa pode votar. (Idade mínima: 16 anos).
// Código para testar no console:
let idade = 15;

if (idade >= 16) {
  console.log("Pode votar!");
} else {
  console.log("Ainda não pode votar.");
}

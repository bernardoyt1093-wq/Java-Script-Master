Trilha JavaScript

Estude os conceitos e veja os exemplos abaixo.

01. Variáveis (let e const)

Usamos para armazenar dados. Let pode mudar, Const não.

let nome = "Bernardo";
const idade = 15;

02. Funções

Blocos que executam tarefas quando chamados.

function msg() {
  return "Olá Mundo!";
}

03. Condicionais (If/Else)

Usado para tomar decisões no código.

if (nota >= 7) {
  console.log("Aprovado");
}

04. Operadores Matemáticos

Cálculos simples de matemática.

let total = (10 + 5) * 2; // Resultado: 30

05. Listas (Arrays)

Uma variável que guarda vários valores.

let lista = ["HTML", "CSS", "JS"];

06. Loops (For)

Repete o código várias vezes.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

07. Objetos

Guarda propriedades de um item.

let player = { nome: "Bernardo", xp: 100 };

08. Eventos

Reage a cliques ou ações do usuário.

botao.addEventListener("click", rodar);

09. Arrow Functions

Funções escritas de um jeito moderno.

const soma = (a, b) => a + b;

10. Manipulação de Strings

Comandos para alterar textos.

let texto = "js";
console.log(texto.toUpperCase());

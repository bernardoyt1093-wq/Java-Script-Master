Trilha JavaScript

Aprenda os fundamentos passo a passo com o Prof. Bernardo.

01. Variáveis (let e const)

Variáveis são como caixas onde guardamos informações. Usamos let para valores que podem mudar e const para valores fixos (constantes).

02. Funções

Funções são blocos de códigos que executam uma tarefa específica. Elas ajudam a evitar repetição, permitindo que você chame o mesmo comando várias vezes.

03. Condicionais (If/Else)

O if serve para o computador tomar decisões. "Se algo for verdade, faça isso; caso contrário (else), faça aquilo". É a base da lógica.

04. Operadores Matemáticos

No JS podemos fazer contas de somar (+), subtrair (-), multiplicar (*) e dividir (/). Também usamos parênteses para definir o que calcular primeiro.

05. Listas (Arrays)

Arrays permitem guardar vários valores em uma única variável. Você acessa os itens pela posição deles, começando sempre do número 0.

06. Loops (For)

O comando for cria uma repetição automática. É útil para percorrer listas ou repetir uma ação várias vezes sem escrever o código repetido.

07. Objetos

Objetos guardam informações detalhadas de uma coisa só. Exemplo: um objeto "Carro" pode ter marca, cor e ano, tudo organizado em propriedades.

08. Eventos

Eventos são ações que acontecem no site, como um clique de botão, o passar do mouse ou o apertar de uma tecla. O JS "escuta" e reage a isso.

09. Arrow Functions

É uma forma moderna e curta de escrever funções no JavaScript. Elas usam o símbolo => e deixam o código mais limpo e fácil de ler.

10. Manipulação de Strings

Strings são textos. No JS, podemos transformar textos para maiúsculas, minúsculas, contar letras ou até trocar palavras dentro de uma frase.
